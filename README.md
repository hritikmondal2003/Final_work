# Final Work - React Portfolio / Project Site

A responsive React-based web application to showcase projects, certifications, and contact information.

## ✨ Features
- Project showcase cards with tags
- Certificate list with individual download buttons
- Contact section with WhatsApp and LinkedIn links
- Responsive layout using Bootstrap

## 🛠 Tech Stack
- React
- React Bootstrap
- React Icons
- Custom CSS

## 🚀 Getting Started

Clone the project:

```bash
git clone https://github.com/hritikmondal2003/Final_work.git
cd Final_work
npm install
npm run dev
